Layout + mediaqueries

Manieren van layout

  • Fixed
  • Fluid

Fixed layout

  • Handig voor papier
  • Handig voor exacte omgevingen
  • Niet geschikt voor het web

Fluid layout

  • Uitstekend geschikt voor alles
  • Ontwerpen in verhoudingen
  • Iets complexer

Flexibele units

  • em
  • rem
  • %
  • vw/vh
Voorbeelden

em

  • De breedte van de letter 'm' (theoretisch)
  • 1em is de standaard font size van de browser
  • Vrijwel altijd 16px
  • Context gevoelig

rem

  • De font size van het 'root element'
  • Niet context gevoelig
  • Support

%

  • Context gevoelig

vw/vh

  • Viewport relative units (width/height)
  • Relatief t.o.v. het initial containing block
  • Support: IE10+ & Chrome20+

Adaptive layouts

  • Fluid layouts
  • Passen zich aan aan viewport
Voorbeeld

Media queries

  • Conditionele CSS
  • @media media type and (condition) ... {}
    • width, height
    • device-width, device-height
    • orientation
    • ...
  • @media (min-width: 400px) {}
  • @media (max-width: 400px) {}
  • Support

Viewport meta

  • <meta name="viewport" content="width=device-width, initial-scale=1.0"/>
