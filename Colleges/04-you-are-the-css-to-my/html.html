Selectors

universal, type, descendant, child, adjacent sibling, attribute

Universal selector

* {
    margin: 0;
    padding: 0;
}

Type selector

body {
    background: white;
}

Descendant selector

article section {
    background: slategray;
}

Direct child selector

fieldset > input {
    background: #fff;
}

Adjacent sibling selector

label + input {
    background: none;
}

Attribute selector

e[attr=value] {
    background: slategray;
}
[for="email"] {
    background: slategray;
}
